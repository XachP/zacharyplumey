# zacharyplumey.com

Personal site for Zachary Plumey — web & lead-gen systems for small business.

## Editing
It's a single static file: **`index.html`** (HTML + CSS inline, no build step).
- Add/swap work items: duplicate a `.card` block in the `#work` section and change the `href`, title, and text.
- Update links/email: search the file for `EDIT:` comments.
- Push to `main` → Render auto-deploys → live on zacharyplumey.com.
